# basic-port-scanner

## Project Overview
This Python project scans ports of a target IP address.

Port scanning helps identify open services running on a system.

## Features
- Scan network ports
- Detect open ports
- Basic networking analysis

## How to Run It
Run the script directly through your terminal or command prompt:

```bash
python basic-port-scanner.py
```

## Concepts Used
- Socket programming
- Networking
- Port scanning

## Language Used
Python

## Purpose
Demonstrates network reconnaissance in cybersecurity.
